Materi 5: Subnetting

← Beranda Materi Sebelumnya → Materi Selanjutnya →

Subnetting membagi jaringan besar menjadi beberapa jaringan kecil. Manfaatnya:

  • Efisiensi penggunaan IP
  • Membatasi broadcast agar tidak mengganggu jaringan lain
  • Keamanan lebih baik karena pemisahan jaringan

Contoh Subnetting

Misal jaringan: 192.168.10.0/24 dibagi menjadi /26 (64 alamat per subnet, 62 host usable):

192.168.10.0/26     (host 1–62)   broadcast 63
192.168.10.64/26    (host 65–126)  broadcast 127
192.168.10.128/26   (host 129–190) broadcast 191
192.168.10.192/26   (host 193–254) broadcast 255

Tips Subnetting

  • Hitung kebutuhan host sebelum menentukan subnet mask
  • Gunakan diagram visual untuk mempermudah perhitungan
  • Catat semua subnet agar tidak terjadi konflik IP

Kesimpulan

Subnetting membantu mengatur jaringan agar lebih efisien, aman, dan mudah dikelola. Penting untuk perencanaan jaringan skala kecil maupun besar.
