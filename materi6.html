Materi 6: Routing Dasar

← Beranda Materi Sebelumnya → Materi Selanjutnya →

Routing adalah proses menentukan jalur dari jaringan sumber ke jaringan tujuan. Dilakukan oleh router dengan membaca tabel routing.

Jenis Routing

  • Statik: rute ditambahkan secara manual oleh administrator
  • Dinamik: router menggunakan protokol seperti RIP, OSPF, EIGRP, BGP untuk update rute otomatis

Contoh Konsep Routing

# contoh pseudo code routing
/* Router A mengetahui:
   - 192.168.10.0/24 via interface 1
   - 192.168.20.0/24 via interface 2 */

Tips Routing

  • Gunakan routing statik untuk jaringan kecil dan sederhana
  • Gunakan routing dinamik untuk jaringan besar dan kompleks
  • Pastikan tabel routing selalu terupdate untuk mencegah loop atau lost packet

Kesimpulan

Routing memastikan data sampai ke tujuan melalui jalur terbaik. Memahami routing statik & dinamik penting untuk manajemen jaringan yang efisien.
