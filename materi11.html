Materi 11: Server & Client

← Beranda Materi Sebelumnya → Materi Selanjutnya →

Model client–server: client meminta layanan, server menyediakan. Contoh: browser (client) ↔ web server (HTTP).

Diagram Client & Server

Jenis Server Umum

  • Web Server (HTTP/HTTPS)
  • File Server (Simpan & share file)
  • Database Server (MySQL, PostgreSQL, SQL Server)
  • DNS Server (Resolusi nama domain)
  • DHCP Server (Alokasi IP otomatis)
  • Mail Server (Email)
  • Application Server (Layanan aplikasi)

Tips Praktis

  • Pastikan server selalu diupdate dan diamankan
  • Batasi akses client sesuai kebutuhan
  • Gunakan backup dan monitoring untuk server kritikal

Kesimpulan

Memahami arsitektur client-server membantu merancang layanan jaringan yang efisien, aman, dan mudah di-maintain.
