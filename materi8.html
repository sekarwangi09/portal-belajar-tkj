Materi 8: Model OSI

Model OSI membagi komunikasi jaringan menjadi 7 lapisan: Physical, Data Link, Network, Transport, Session, Presentation, dan Application. Tujuannya untuk standarisasi komunikasi agar perangkat berbeda dapat saling terhubung.

Diagram Model OSI

Ringkasan Fungsi Lapisan

7. Application   : HTTP, SMTP, FTP
6. Presentation  : Enkripsi, Kompresi
5. Session       : Manajemen sesi
4. Transport     : TCP/UDP, Port
3. Network       : IP Routing
2. Data Link     : MAC, Switch
1. Physical      : Media, Sinyal

Tips Memahami OSI

  • Ingat urutan lapisan dari Physical sampai Application
  • Hubungkan tiap lapisan dengan perangkat/layanan nyata
  • Gunakan diagram untuk memvisualisasikan flow data

Kesimpulan

Model OSI mempermudah pemahaman komunikasi jaringan, troubleshooting, dan interoperabilitas antar perangkat. Setiap lapisan memiliki peran spesifik agar data dapat dikirim secara efisien dan aman.
