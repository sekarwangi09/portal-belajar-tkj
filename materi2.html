Materi 2: Perangkat Jaringan

Perangkat jaringan menghubungkan host agar data dapat berpindah. Perangkat yang umum digunakan di jaringan adalah:

  • Router: menghubungkan jaringan berbeda & melakukan routing data antar jaringan.
  • Switch: menghubungkan host dalam satu LAN, cerdas menggunakan MAC table.
  • Access Point: menghubungkan perangkat nirkabel (Wi-Fi) ke jaringan.
  • Firewall: menyaring trafik untuk keamanan jaringan.
  • Modem: modulasi dan demodulasi sinyal untuk koneksi ISP.
  • Hub: perangkat sederhana untuk menghubungkan beberapa host dalam satu LAN (tidak cerdas).
  • Repeater: memperkuat sinyal agar jangkauan jaringan lebih luas.

Fungsi Perangkat Jaringan

  • Router: mengatur jalur data antar jaringan dan akses internet.
  • Switch: menghubungkan host dalam satu jaringan LAN secara efisien.
  • Access Point: menyediakan koneksi Wi-Fi untuk perangkat nirkabel.
  • Firewall: mencegah akses yang tidak sah dan mengamankan jaringan.
  • Modem: menjembatani jaringan lokal dengan ISP.

Tips Penggunaan Perangkat

  • Letakkan router dan switch di lokasi strategis untuk jangkauan optimal.
  • Gunakan firewall untuk memproteksi jaringan dari ancaman eksternal.
  • Gunakan Access Point dengan SSID dan password aman.
  • Monitor trafik jaringan untuk menghindari bottleneck.

Kesimpulan

Perangkat jaringan memiliki fungsi spesifik untuk menjaga koneksi, keamanan, dan performa jaringan. Memahami tiap perangkat penting untuk membangun jaringan yang efektif dan aman.
