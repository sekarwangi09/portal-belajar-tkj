Sistem Operasi (OS) adalah perangkat lunak inti yang mengelola hardware dan menjalankan aplikasi. Contoh: Windows, Linux, macOS.

Jenis Software

  • Aplikasi: Office, Browser, Media Player
  • Utility: Antivirus, Backup, Disk Cleanup
  • Driver: Menghubungkan OS dengan hardware

Diagram OS & Software

Tips Mengelola Software

  • Install OS sesuai hardware
  • Gunakan software resmi atau aman
  • Rutin update OS dan aplikasi

Kesimpulan

Software & OS bekerja sama untuk menjalankan komputer secara optimal. OS mengelola hardware, software menjalankan tugas pengguna.
