HTML: Struktur Dasar

HTML digunakan untuk membuat struktur halaman web. Contoh:

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Judul</title>
</head>
<body>
  <h1>Halo Dunia</h1>
  <p>Paragraf pertama</p>
</body>
</html>

CSS: Styling Dasar

CSS digunakan untuk memberi style pada halaman. Contoh:

body { font-family: Arial; color:#333; }
h1 { color: #5b7cfa; }
p { line-height: 1.6; }
